# Arkan Help

Contextual help system for Frappe v16. Provides in-app help topics, multilingual content, and context-aware help mapping for DocTypes, pages, and reports.

## Features

- **Help Topics** — Organized help entries with machine-readable keys
- **Multilingual Content** — Markdown, HTML, video, and interactive content per language
- **Context Mapping** — Map help to specific DocTypes, fields, views, and actions
- **Role-Based Visibility** — Show help content based on user roles
- **Global Settings** — Configure help behavior (field icons, navbar, caching)

## Installation

```bash
bench get-app arkan_help
bench --site <site_name> install-app arkan_help
```
