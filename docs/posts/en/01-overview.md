# ❓ Arkan Help — Contextual Help System for Frappe Apps

> **Contextual Help System for Frappe Apps** — Open source, self-hosted, integrated with ERPNext.

---

## Why Arkan Help?

Businesses in the Documentation space face real challenges:
- ❌ Users don't know how to use Frappe forms
- ❌ Training costs for new employees
- ❌ No inline help system in Frappe
- ❌ Help content not role-specific

**Arkan Help solves all of this** — natively inside Frappe/ERPNext.

## ✨ Key Features

- ✅ Field-Level Help Icons (ⓘ)
- ✅ Navbar Help Button (❓)
- ✅ 6-Level Specificity Resolution
- ✅ File-Based Help (Markdown)
- ✅ Role-Specific Help Content
- ✅ Bilingual (Arabic/English) Support
- ✅ Help Analytics & Tracking
