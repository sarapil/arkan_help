## يعمل مع ERPNext الحالي لديك

Arkan Help لا يستبدل ERPNext — بل **يُوسّعه**. ثبّته بجانب إعدادك الحالي:

```bash
bench get-app https://github.com/sarapil/arkan_help
bench --site your-site install-app arkan_help
```

كل بياناتك وسير عملك وتخصيصاتك الحالية تبقى كما هي.
