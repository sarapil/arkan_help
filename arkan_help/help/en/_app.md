# Arkan Help

Contextual help system.

## Key Features

- Comprehensive management and tracking
- Role-based access control via CAPS
- Full Arabic + English support
- Integration with ERPNext ecosystem

## Getting Started

1. Navigate to the **Arkan Help** workspace from the sidebar
2. Follow the onboarding wizard for initial setup
3. Use the ❓ help icon on any form for contextual guidance
